OntoQAV WebVOWL Plugin

To run this plugin in WebVOWL:

1. copy the code below to WebVOWL's index.html to initialise.
2. update source file location for jquery-3.2.1.js and ontoqav.js

<!-- OntoQAV Plugin -->
	
<script src="../OntoQAV/js/ontoqav.js"></script>
	
<script src="../OntoQAV/js/jquery-3.2.1.js"></script>
	
<script>initOntoQAV();</script>
